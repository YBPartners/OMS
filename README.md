## v14.0 신규 기능 — 웹 푸시 + 데이터 관리 + 매출/정산 차트

### 웹 푸시 알림 (Service Worker)
- **Service Worker 등록**: `/sw.js` — 오프라인 캐시 + 푸시 알림 수신
- **로컬 알림**: 새 알림 증가 시 자동 브라우저 푸시 알림 표시
- **알림 클릭**: 알림 클릭 시 해당 페이지로 이동
- **구독/해제 API**: `POST /api/system/push/subscribe`, `/push/unsubscribe`

### 데이터 임포트/백업 자동화
- **데이터 임포트**: CSV/JSON → DB 일괄 입력 (주문/사용자/조직/수수료, 최대 500행)
  - 신규 삽입 / Upsert 모드 지원
  - CSV 파일 자동 파싱 (UTF-8, BOM 제거)
- **스냅샷 백업**: 전체 DB 데이터 → JSON 파일 다운로드
- **스냅샷 복원**: JSON 파일 → DB 데이터 복원 (선택적 기존 삭제)
- **시스템 관리 4탭**: 세션관리 / DB현황 / 데이터임포트 / 백업복원

### 대시보드 추가 차트 (HQ/REGION)
- **일별 매출 추이**: 30일간 매출액/지급액/주문건수 복합 라인 차트
- **지역별 매출 비교**: 수평 바 차트 (매출액 vs 지급액)
- **정산 현황 요약**: 산출/확정/지급 카드 + 도넛 차트
- **지역별 정산**: 스택 바 차트 (확정/지급/산출)
- **최근 정산 Run**: 테이블 (Run ID, 기간, 상태, 건수, 금액)

### 신규 API 엔드포인트 (8개)
| 엔드포인트 | 메서드 | 설명 |
|-----------|--------|------|
| `/api/stats/revenue-trend` | GET | 일별/월별/지역별 매출 추이 |
| `/api/stats/settlement-summary` | GET | 정산 현황 (상태/지역/Run별) |
| `/api/system/import/:table` | POST | 데이터 임포트 (JSON 배열) |
| `/api/system/snapshot` | GET | 스냅샷 백업 (JSON) |
| `/api/system/snapshot/restore` | POST | 스냅샷 복원 |
| `/api/system/push/subscribe` | POST | 푸시 알림 구독 |
| `/api/system/push/unsubscribe` | POST | 푸시 알림 해제 |
| `/sw.js` | - | Service Worker (오프라인+푸시) |

## v13.0 신규 기능 — 시스템 관리 + 보안 강화 + 글로벌 검색 + 타임라인

### 시스템 관리 (SUPER_ADMIN 전용)
- 시스템 정보 대시보드: 버전, 사용자/주문/조직/세션 통계
- 활성 세션 관리: 목록 조회, 개별 강제종료, 전체 초기화
- 데이터베이스 현황: 34개 테이블 행 수 그리드
- 데이터 내보내기: 화이트리스트 기반 테이블 데이터 export

### 보안 강화
- **계정 잠금**: 5회 연속 로그인 실패 → 5분간 잠금 (423 응답)
- **비밀번호 정책**: 최소 6자 + 영문+숫자 필수 + 이전 비밀번호 재사용 불가
- 감사 로그: LOGIN_FAILED, SESSION_REVOKE, SESSION_PURGE, DATA_EXPORT

### 글로벌 검색 (Cmd+K / Ctrl+K)
- 주문/사용자/조직 통합 검색 (2자 이상, 300ms debounce)
- 결과 타입별 아이콘/배지/금액 표시
- 검색 결과 클릭 시 해당 페이지 이동

### 주문 타임라인 (활동 이력)
- 주문 전체 이력 통합 조회 (6개 소스: 감사로그, 배분, 배정, 보고서, 검수, 정산)
- 타임라인 모달: 이벤트별 아이콘/색상, 시간순 목록
- 글로벌 검색에서 주문 클릭 시 자동 표시
